# Cainiao1992 Tap

## How do I install these formulae?

`brew install cainiao1992/tap/<formula>`

Or `brew tap cainiao1992/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cainiao1992/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
